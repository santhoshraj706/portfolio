# Santhosh Raj T — Portfolio

A high-fidelity, professional portfolio designed for the modern web. Built with specialized focus on full-stack development, AI integration, and cybersecurity.

## 🚀 Key Features

- **Interactive UI**: Glassmorphism, neon accents, and professional animations.
- **Resend Contact Form**: Fully functional serverless backend using Vercel & Resend API.
- **Slate Theme**: A sophisticated, mid-tone aesthetic (not too dark, not too light).
- **Responsive Design**: Optimized for all devices from mobile to desktop.
- **Embedded Resume**: Direct access to professional credentials.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js (Vercel Serverless Functions)
- **Email Service**: Resend API
- **Deployment**: Vercel

## 📦 Local Development

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory:
   ```env
   RESEND_API_KEY=your_resend_api_key
   ```
4. Run locally with Vercel CLI:
   ```bash
   npx vercel dev
   ```
